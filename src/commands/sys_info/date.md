# date

* `date`
  * 作用：在终端中显示当前时间
  * 举例
    * 显示时间
        ```bash
        # date
        Wed Aug 28 17:01:10 CST 2019
        ```
    * 加上格式
        ```bash
        # date +%Y%m%d_%H:%M:%S
        20201031_12:04:49
        ```
    * 如果格式中间有空格，则需要加上双引号
        ```bash
        # date "+%Y%m%d %H:%M:%S"
        20201031 12:03:57
        ```
